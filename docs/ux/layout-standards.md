# Layout Standards

Version: 1.0
Scope: Cross-screen layout rules and spacing rhythm for Haven.

## Spacing Rhythm

These are the canonical values. Screen specs may override with justification.

- Horizontal page padding: 16dp
- Major section gap: 24dp
- Minor element gap: 12dp
- Bottom content padding above nav: at least 16dp
