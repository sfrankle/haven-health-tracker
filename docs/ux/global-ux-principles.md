
# Copy Guidelines
Use:
- "Add entry"
- "Saved."
- "Recent"
- "Low energy"
- "Try this"
- "Start"

Avoid:
- "Great job"
- "You missed yesterday"
- "You should"
- "Fix yourself"
- "You're behind"

# Layout Specification
- Bottom navigation remains visible.
- No dense top controls.
- Keep visible choices concise.

## Spacing Rhythm
- Horizontal page padding: 16dp
- Major section gap: 20-24dp
- Item gap within section: 8-12dp

# Accessibility
- Touch targets meet Material guidance
- Icons paired with labels
- Contrast meets AA targets in light and dark themes

# Acceptance Criteria
- No streak indicators or pressure copy are shown
- Bottom navigation remains visible on all screens
- Visual spacing follows defined rhythm and avoids dense form-like layouts
